# GymBooking Spring Boot - REST API

A gym booking management system built with Spring Boot, JPA/Hibernate and MySQL.

## Setup

1. Clone the repository
2. Copy `src/main/resources/application.properties.example` to `src/main/resources/application.properties`
3. Fill in your database credentials
4. Run the application

## Technologies
- Java 21
- Spring Boot 3.5
- Spring Data JPA / Hibernate
- MySQL
- Maven
